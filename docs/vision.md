# Simply BTP — Vision

## Mission

Devenir la **référence européenne** des applications terrain pour le suivi de chantier BTP.

Simply est le compagnon mobile/tablette des équipes sur chantier. Operis (futur) pilote depuis le bureau.

## Problème résolu

Sur un chantier, les conducteurs perdent du temps à :
- Chercher l'info (plans, photos, CR)
- Saisir deux fois la même donnée (papier + Excel)
- Coordonner sans outil adapté au terrain

Simply centralise tout **par chantier**, en **moins de 3 clics**.

## Utilisateurs

| Persona | Besoin principal |
|---------|------------------|
| Conducteur de travaux | Vue globale, retards, budget, CR |
| Artisan / sous-traitant | Tâches, photos, pointage |
| Architecte / MOE | Plans, réserves, PV |
| Entreprise générale | Budget, planning, certificats |

## Principes produit

1. **Terrain-first** — tablette, gants, soleil
2. **Gain de temps réel** — chaque feature mesurable
3. **Simplicité** — zéro feature gadget
4. **Premium** — crédible face à Procore
5. **Données reliées** — tout lié au chantier

## Positionnement

```
Operis  = bureau, multi-projets, appels d'offres
Simply  = terrain, 1 chantier actif, exécution
```

## Succès v1

- 10 modules fonctionnels
- Utilisable sans formation
- Déployé Vercel, responsive tablette
- Base évolutive vers Supabase + Operis

## North Star

Temps moyen pour documenter une journée de chantier : **< 2 minutes**.
