# Simply BTP

Application de gestion de chantier BTP, pensée **terrain-first** (tablette et mobile en priorité).

## Stack

- **Next.js 16** (App Router, Turbopack)
- **TypeScript**
- **Tailwind CSS v4**
- **Supabase** (EU/Paris — branché plus tard)

## Commandes

```bash
# Développement local
npm run dev

# Build de production
npm run build

# Démarrer le build en local
npm start

# Lint
npm run lint
```

## Déploiement Vercel

```bash
npx vercel          # première fois : login + link
npx vercel --prod   # déploiement production
```

Ou connectez le dépôt GitHub à [vercel.com](https://vercel.com) pour des déploiements automatiques.
